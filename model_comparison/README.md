# 模型对比分析模块

## 功能概述

模型对比分析模块用于对比多个系统自有模型的性能，支持自动超参数优化，帮助用户选择最佳模型。

## 核心功能

### 1. 模型对比
- **支持的系统模型**：线性分类器、三层感知机、多层感知机、CNN、CNN-Transformer
- **对比指标**：
  - 测试准确率
  - 训练时间
  - 泛化得分（训练集与测试集准确率差异）
  - 矿点检测率

### 2. 数据加载
- **输入格式**：H5格式的地球化学数据集
- **矿点文件支持**：CSV、Excel格式
- **两种加载模式**：
  - 分别指定训练/测试矿点文件
  - 单文件自动分割（可设置测试集比例）

### 3. 数据配置
- 内部验证比例：0.1-0.5（默认0.2）
- 缓冲区半径：0-5000米（默认500米）
- 窗口大小：8-256像素（默认64）
- 步长：1-256像素（默认32）

### 4. 训练配置
- 训练轮次：1-500（默认50）
- 批量大小：1-256（默认32）
- 学习率：0.00001-1.0（默认0.001）
- 优化器：Adam、AdamW、SGD、RMSprop

### 5. 自动优化（AutoML）
- 自动搜索最优超参数
- 支持系统内置模型的统一比较与自动优化
- 可设置优化次数：10-200次（默认50次）
- 自动生成PDF优化报告

### 6. 结果分析
- **可视化图表**：
  - 测试准确率对比柱状图
  - 训练时间对比柱状图
  - 泛化得分对比柱状图
  - 矿点检测率对比柱状图
- **结果导出**：CSV格式的详细对比表格

## 使用流程

1. **加载数据**
   - 选择完整数据集H5文件
   - 选择矿点文件（训练/测试）
   - 配置数据参数
   - 点击"加载数据"按钮

2. **模型对比**
   - 勾选要对比的模型
   - 设置训练参数
   - 点击"开始对比"按钮
   - 查看实时训练日志

3. **查看结果**
   - 切换到"结果分析"标签页
   - 选择不同图表类型查看对比结果
   - 导出CSV格式的详细结果

4. **自动优化**
   - 切换到"自动优化"标签页
   - 选择要优化的模型
   - 设置优化次数
   - 点击"开始自动优化"
   - 导出PDF优化报告

## 数据要求

### H5数据集格式
- 必须包含地球化学元素数据
- 支持多通道输入
- 自动检测数据模式

### 矿点文件格式
- **必需列**：X坐标、Y坐标
- **支持的列名**：
  - X坐标：x, coord_x, point_x, east, easting, 东坐标, 横坐标
  - Y坐标：y, coord_y, point_y, north, northing, 北坐标, 纵坐标
- **可选列**：标签、类别（用于分层采样）

## 输出结果

### 对比结果表格
包含以下指标：
- 模型名称
- 训练准确率
- 验证准确率
- 测试准确率
- 训练时间
- 泛化得分
- 矿点检测率

### AutoML报告
- 每个模型的最优超参数
- 验证集准确率
- 超参数搜索过程
- PDF格式报告

## 注意事项

1. 数据加载后，修改配置参数需要重新加载数据
2. 模型训练过程中可以随时停止
3. 建议先用较少的训练轮次测试，确认无误后再进行完整训练
4. AutoML优化时间较长，建议合理设置优化次数
5. 确保矿点坐标与H5数据集的坐标系统一致

## 技术特点

- 多线程异步处理，界面不卡顿
- 自动内存管理，防止内存溢出
- 支持GPU加速训练
- 实时显示训练进度和日志
- 完整的错误处理和提示
